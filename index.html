Mathematical Background:

The color transformation on this site is achieved through a linear interpolation (lerp) algorithm based on temperature values. The temperature data, obtained from the OpenWeatherMap API in Kelvin, is converted to Celsius by subtracting 273.15.

The normalized temperature, ranging from -40°C to +40°C, is calculated by mapping the temperature to a scale between 0 and 1. This normalized value serves as a parameter for linearly interpolating between the selected cooler and warmer colors.

Linear Interpolation Formula:

lerp (a,b,t) = a ⋅ ( 1 − t ) + b ⋅ t

Where:
a is the cooler color,
b is the warmer color,
t is the normalized temperature.

Color Transformation:

Based on the temperature in Stockholm, the background color smoothly blends between the cooler and warmer colors based on the linear interpolation. Colder temperatures move it towards the user’s chosen color for ”COLD”, while warmer temperatures move it toward the user's chosen color for ”WARM”.

The RGB code is displayed on the page, giving the users the exact color code for the temperature-color-mix.
